Highlight.js

What is highlight.js?

Highlight.js is a syntax highlighter written in JavaScript, works in the browser and server.

More specific, it makes your code alive with color on browsers.

Examples: this page or Create QR Code with Python

More about it at: highlightjs.org

Installtaion:

  1. Add this to your <head> </head> 

    2. <link rel="stylesheet" 
href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.7.0/styles/dark.min.css">

  2. In <body> </body>, add 

    3. <pre><code class="language-html"> Your code goes here.
</code> </pre>

    And: 

    <script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.7.0/highlight.min.js"> </script>
<script> hljs.highlightAll(); </script>

More theme at: cdnjs.com/libraries/highlight.js

Change the class="language-html" into your code language. For example: class="language-cpp"
