Create QR Code with Python

Requirements:

  • PyQRCode Lib.

  • Pyzbar Lib.

  • PIL Lib.

  • Installs Lib: 

  pip install <lib name in lowercase>

Code: 

import pyqrcode
from pyzbar.pyzbar import decode
from PIL import Image
                
# Create a QR code
qr = pyqrcode.create("https://www.youtube.com/watch?v=exI40Lqyd0k")
qr.png("test.png", scale=9)
               
# Read a QR code
d = decode(Image.open("qrcode.png"))

Usage:

  • Copy this code to your python file and run it, it will create an image names qrcode.

  • The youtube example link will show up when scanning the image (can be link or text).
